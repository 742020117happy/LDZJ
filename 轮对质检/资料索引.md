## 三、CGXi-Robot（协作机器人/机械臂）— 对应代码 `CGXi_Remote`

> 主程序通过 **Modbus TCP** 与 CGXi 机械臂控制器通信，完整的 Modbus 地址映射在 `s_CGXi_Date`（`Variable.h:80-207`）。
> Modbus 客户端封装在 `ModeBusTCP_Client.h/cpp`，业务逻辑在 `CGXi_Remote.h/cpp`。
> SDK头文件 `robotapi.h` 和 `basestruct.h` 已复制到主程序目录 `Robot_App/` 下供代码引用。

## 五、主程序架构一览 (`Robot_App/`)

```
Robot_App/
├── main.cpp                # Qt 应用入口
├── Robot_App_Widget.h/cpp  # 主窗口，调度所有模块（Init/Scan/DB绑定）
├── Variable.h/cpp          # 全局类型定义、数据结构（单例模式）
│   ├── s_CGXi_Date         #  CGXi Modbus寄存器完整映射（1000 reg）
│   ├── s_CGXi_DB           #  CGXi 全局数据库（含联合体读写缓存）
│   ├── Magic_WriteData     #  Magic 写入数据结构体
│   ├── Magic_ReadData      #  Magic 读取数据结构体
│   └── s_Magic_DB          #  Magic 全局数据库
├── CGXi_Remote.h/cpp       #  机械臂控制业务（Modbus TCP → 线圈/保持寄存器读写）
├── ModeBusTCP_Client.h/cpp #  Modbus TCP 底层客户端封装
├── Magic_Remote.h/cpp      #  AMR底盘控制业务（HTTP REST API → GET/POST/PUT/DELETE）
├── HTTP_Client.h/cpp       #  HTTP 底层客户端封装（基于QNetworkAccessManager）
├── Magic_Map_View.h/cpp    #  地图可视化组件
├── Robot_Client.h/cpp      #  TCP/JSON 通用客户端（Socket通讯）
├── Robot_Server.h/cpp      #  TCP Server（供外部设备连接）
├── Scan_Client.h/cpp       #  UDP 精扫设备客户端
├── Fr_Light.h/cpp          #  前后灯控制
├── robotapi.h              #  CGXi SDK 头文件（从sdk复制）
├── basestruct.h            #  CGXi SDK 数据结构头文件
├── System_DB/              #  系统配置
│   └── Communicate_DB.json  #  通讯参数配置数据库
├── Work_DB/                #  工作数据库
│   └── Axis_Position.json   #  轴位置预设数据
├── Images/                 #  UI图片资源
└── Video_DB/               #  视频数据目录（空，预留）
```
